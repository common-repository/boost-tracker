== Description ==
The plug in allows you to track the performance of your Boost campaigns. More info: http://boost.re

Is your content eligible to be boosted? Check the guidelines to find out: http://blog.boost.re/guidelines


== Installation ==
Upload the .zip to your server and enable the plugin. Then, go to the post you want to Boost and add this text at the end of the post:

[boost-tracker] 

That is all!
